# 说明
- 支持全国所有地区
- 抓包教程：安装完mitmproxy后，打开cookie.bat，然后 PC 微信进入约苗`公众号` -> 打疫苗 -> 疫苗预约 跳出地区选择后关闭 -> 我的 -> 授权登录 
即可自动保存header
- 双击miaomiao.exe，即可运行
